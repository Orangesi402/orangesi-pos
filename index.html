กำลังเชื่อมต่อ...
ข้อมูลบันทึกในเครื่องนี้ทันที และซิงค์ขึ้น Firebase Cloud อัตโนมัติ — ทุกเครื่องเห็นข้อมูลตรงกัน
⚙️ ยังไม่ได้ตั้งค่า Firebase — ซิงค์ข้ามเครื่องยังไม่ทำงาน
วิธีตั้งค่า (ทำครั้งเดียว ~5 นาที):
1. เปิด console.firebase.google.com → ล็อกอินด้วย Google
2. กด "Add project" → ตั้งชื่อโปรเจกต์ → Create project
3. กด "</>" (Web app) → ตั้งชื่อแอป → Register app
4. คัดลอก firebaseConfig ที่ได้ → วางแทนค่า PASTE_..._HERE ใน index.html
5. เมนูซ้าย กด Build → Realtime Database → Create database → เลือก "Start in test mode" → Enable
6. อัปโหลด index.html ใหม่ขึ้น GitHub → เสร็จ ✅
